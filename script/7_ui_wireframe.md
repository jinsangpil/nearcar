UI Wireframe.md - 니어카

본 문서는 니어카 통합 플랫폼의 주요 화면 설계를 정의합니다.
Mobile First 전략을 기본으로 하며, 운영자(Admin) 페이지는 Desktop 환경을 기준으로 작성되었습니다.

1. 고객(Client) - 모바일 웹

1.1 메인 & 신청 1단계 (차량 정보 입력)

목표: 사용자가 가장 먼저 "내 차 정보"를 입력하게 하여 이탈률 감소

+-------------------------------------------------------+
| [Logo] 니어카                    [My] [Menu]          |
+-------------------------------------------------------+
|                                                       |
|   중고차 살 때,                                       |
|   전문가와 동행하세요!                                |
|                                                       |
|   +-----------------------------------------------+   |
|   |  [Tab] 차량번호로 조회  |  모델로 찾기 (Select)  |   |
|   +-----------------------------------------------+   |
|   |                                               |   |
|   |  [ 차량번호 입력 (예: 12가3456)         ]      |   |
|   |                                               |   |
|   |  [   조회하기 (Button)                  ]      |   |
|   |                                               |   |
|   +-----------------------------------------------+   |
|                                                       |
|   [Banner] 누적 진단 10,000건 돌파!                   |
|   [Review] "기사님이 정말 꼼꼼해요" ⭐⭐⭐⭐⭐          |
|                                                       |
+-------------------------------------------------------+
| [Footer] 사업자 정보 | 이용약관                        |
+-------------------------------------------------------+


Interaction Note:

차량번호 조회: 국토부 API 연동 → 성공 시 차량 기본정보(모델, 연식) 노출하며 Step 2 이동.

모델로 찾기: 제조사(현대) > 모델(그랜저) > 상세모델(IG) 순차적 Select Box 노출 (Cascading).

1.2 신청 2단계 (옵션 선택 및 견적 확인)

TRD v1.2 반영: 선택한 지역과 차량 등급에 따라 가격이 실시간으로 변동됨

+-------------------------------------------------------+
| [<-]  진단 옵션 선택                                   |
+-------------------------------------------------------+
|                                                       |
|  **1. 차량 정보 확인** |
|  더 뉴 그랜저 IG (2020) / 대형 / 국산                  |
|                                                       |
|  **2. 지역 선택** (출장비 계산용)                       |
|  [ 시/도 선택 (v) ]  [ 시/구/군 선택 (v) ]             |
|  * 경기 가평군은 출장비 20,000원이 추가됩니다.          |
|                                                       |
|  **3. 패키지 선택** |
|  (o) **Basic** (기본 진단)             55,000원       |
|  ( ) **Pro** (하부/엔진 정밀)          +30,000원      |
|                                                       |
|  ---------------------------------------------------  |
|  **4. 일정 선택** |
|  [ 2025-12-05 (금) ]                                  |
|  [ 10:00 ] [ 14:00 ] [ 16:00 ]                        |
|                                                       |
+-------------------------------------------------------+
|  **예상 결제 금액** |
|  기본 55,000 + 대형할증 10,000 + 지역 20,000           |
|  Total: **85,000원** |
|  [         결제하고 신청하기 (Sticky Bottom)    ]      |
+-------------------------------------------------------+

2. 기사(Inspector) - 모바일 앱/웹

2.1 대시보드 (메인)

오늘의 일정과 수락 대기 중인 배정 건을 최상단에 노출

+-------------------------------------------------------+
| [Menu]  김기사님, 오늘도 안전운전!                     |
+-------------------------------------------------------+
|                                                       |
|  **배정 요청 (1건)** [New]                           |
|  +-----------------------------------------------+    |
|  | 벤츠 E-Class (W213) | 서울 강남구 역삼동        |    |
|  | 12.05(금) 14:00    | 예상수익 45,000원        |    |
|  | [ 거절 ]              [ 수락하기 ]             |    |
|  +-----------------------------------------------+    |
|                                                       |
|  **오늘의 일정** |
|  - 10:00 아반떼 CN7 (진단완료)                        |
|  - 16:00 그랜저 IG (이동중 -> 도착확인)               |
|                                                       |
|  **이번 달 실적** |
|  완료 15건 | 정산예정 1,200,000원                      |
|                                                       |
+-------------------------------------------------------+


2.2 진단 체크리스트 작성

현장에서 한 손으로 빠르게 입력 가능하도록 토글/버튼 위주 설계

+-------------------------------------------------------+
| [<-]  진단 작성 (12가3456)                  [임시저장]|
+-------------------------------------------------------+
|                                                       |
|  **1. 외관 (Exterior)** |
|  - 전면 유리:  [양호] [찍힘] [금감]                   |
|  - 운전석 도어: [양호] [스크래치] [판금]              |
|  [ + 사진 추가 (카메라 구동) ]                         |
|                                                       |
|  **2. 엔진룸 (Engine)** |
|  - 엔진 오일:  [양호] [부족] [누유]                   |
|  - 냉각수:    [양호] [오염] [누수]                   |
|  [ + 사진 추가 ]                                      |
|                                                       |
|  ... (하부, 실내, 편의장치 등) ...                     |
|                                                       |
|  **종합 의견** |
|  [ 텍스트 입력 (음성인식 권장)                  ]      |
|                                                       |
|  [              최종 제출하기                       ]  |
+-------------------------------------------------------+

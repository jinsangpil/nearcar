# Site Map (Page List) - 니어카

본 문서는 니어카 통합 플랫폼의 전체 예상 페이지 리스트입니다.

Client는 카바조의 사용자 경험을 벤치마킹하였으며, Inspector는 모바일 현장 업무 최적화, Admin은 데이터 관리 편의성을 중심으로 구성되었습니다.

---

## 1. 고객 (Client) - Mobile/PC Web

> **벤치마킹 포인트 (Carvazo):** 신뢰감을 주는 랜딩, 단계별(Step-by-step) 신청 프로세스, 직관적인 레포트 뷰어

### 1.1 공통 / 랜딩

| Page ID | URL | 화면명 | 주요 기능 |
|---------|-----|--------|----------|
| C-01 | `/` | 메인 (Landing) | 서비스 특장점, 누적 진단 수, 후기 하이라이트, 차량번호 조회(CTA) |
| C-02 | `/service` | 서비스 소개 | 진단 범위 설명, 기사 검증 시스템 소개, 이용 가이드 |
| C-03 | `/pricing` | 이용 요금 | 차종/지역별 예상 요금표 (동적 가격 정책 안내) |
| C-04 | `/reviews` | 실제 후기 | 별점 및 포토 후기 리스트, 필터링 |
| C-05 | `/faq` | 자주 묻는 질문 | 결제, 환불, 당일예약 등 FAQ |

### 1.2 진단 신청 (Booking Flow) - Core

| Page ID | URL | 화면명 | 주요 기능 |
|---------|-----|--------|----------|

## 2. 기사 (Inspector) - Mobile Web/App

> **설계 포인트:** 현장에서 한 손으로 조작 가능한 UI, 직관적인 스케줄 관리

### 2.1 인증 및 대시보드

| Page ID | URL | 화면명 | 주요 기능 |
|---------|-----|--------|----------|
| I-01 | `/inspector/login` | 기사 로그인 | 아이디/비밀번호 로그인 |
| I-02 | `/inspector` | 대시보드 | 오늘의 일정 요약, 신규 배정 요청 건수, 이달의 수익 |
| I-03 | `/inspector/schedule` | 일정 캘린더 | 월간/주간 뷰, 예약된 일정 확인 |

---

### 2.2 배정 및 진단 업무 - Core

| I-13 | `/inspector/photo/{id}` | 사진 업로드 | 부위별 가이드라인에 맞춰 촬영 및 업로드 |
| I-14 | `/inspector/preview/{id}` | 레포트 미리보기 | 제출 전 최종 확인, 최종 제출 버튼 |

## 3. 운영자 (Admin) - PC Web

> **설계 포인트:** 많은 데이터를 한눈에 보는 테이블 뷰, 강력한 필터링, 데이터(차량/가격) 제어권

### 3.1 대시보드

| Page ID | URL | 화면명 | 주요 기능 |
|---------|-----|--------|----------|
| A-01 | `/admin/login` | 관리자 로그인 | 보안 로그인 (2FA 권장) |
| A-02 | `/admin` | 통합 대시보드 | 오늘의 신청/배정 현황, 매출 통계 그래프, 미배정 경고 알림 |

### 3.2 서비스 운영 (Operations) - Core
